JS 개요

스크립트 언어(script)

        기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어

        별도로 소스 코드를 컴파일 하지 않고,

        interpreter를 이용하여, 소스코드를 한 줄씩 읽어 바로 실행한다.
        -> 컴파일 소요시간 X
        -> 프로그램 실행시간 느림

        -> 짧은 소스코드 파일로 상호작용 하도록 고안됨

자바 스크립트(JavaScript / JS )란? 

        웹 브라우저에서 많이 사용하는 interpreter 방식의 객체 지향 프로그래밍

        ECMA Script 표준을 따르는 대표적인 웹 기술
        Ecma Script? = . 자바스크립트를 표준화하기 위해 만들어진 언어

자바스크립트 작성 방법

  • 브라우저 콘솔에 직접 작성
  • html 내부에 script 태그를 이용해서 작성하는 방법(internal)
  • html 외부에 (.js)파일을 이용해서 작성(external)
  • 태그에 직접 JS코드를 작성(inline)
